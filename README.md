# 📚 Vokabeltrainer

Ein im Rahmen der Oberstufe entwickelter Vokabeltrainer zur Unterstützung beim Erlernen und Wiederholen von Fremdsprachenvokabeln für Kinder der Unter- und Mittelstufe.

## Projektinformationen

Dieses Projekt wurde ursprünglich **nicht über GitHub entwickelt**. Die Entwicklung und Bereitstellung erfolgten über die schulische Infrastruktur. Aus diesem Grund dient dieses Repository hauptsächlich als **Sammlung und Archivierung der Projektdateien**.

## Funktionen

- Verwaltung von Vokabeln
- Abfrage- und Lernmodus
- Unterstützung beim selbstständigen Lernen
- Benutzerfreundliche Oberfläche
